# 🎯 CS2 Skin Sniper — монитор редких скинов + авто-скупка

Telegram-бот, который следит за площадками CS2, ловит **дешёвые лоты, редкие
паттерны (paint seed), низкий float и редкие наклейки** по твоим правилам, шлёт
алерты и **автоматически покупает** подходящее на твой аккаунт — с защитой от
слива баланса.

- **DMarket** — мониторинг **и авто-покупка** по официальному Trading API.
- **Steam Market** — только мониторинг/алерты (у Steam нет API покупки;
  автоматизация сайта нарушает правила Steam, поэтому автобай там не делается).

> ⚠️ **Авто-покупка тратит РЕАЛЬНЫЕ деньги с твоего баланса DMarket.** Поэтому по
> умолчанию включён **Dry-run** (бот только пишет «купил бы», ничего не тратя) и
> **автопокупка выключена**. Сначала протестируй, потом включай реальные траты.

---

## Возможности

- 📏 **Правила поиска**: название/часть, макс. цена, диапазон float, редкие paint
  seed, ключевые слова наклеек, мин. скидка к реф-цене («дёшево»).
- 🔔 **Алерты** в Telegram по каждому найденному лоту.
- 🛒 **Авто-покупка** на DMarket по правилу (если включена).
- 🛡 **Защита**: dry-run, лимит цены за предмет, дневной лимит трат, гвард
  «не тратить баланс ниже N», антидубли (один лот — один раз).
- 💰 Баланс, 🧾 журнал покупок, 📊 статус, переключатели прямо в боте.
- 🧪 **Mock-маркет** для теста всей логики без денег и ключей.

## ⚙️ Установка

### 1. Бот и твой ID
- Токен: [@BotFather](https://t.me/BotFather) → `/newbot`.
- Твой ID: [@userinfobot](https://t.me/userinfobot).

### 2. Ключи DMarket (для авто-покупки)
Личный кабинет DMarket → **Trading API** → создай **Public key** и **Secret key**.
(Только для мониторинга ключи не нужны.)

### 3. Настройка
```bash
cd cs2_skin_sniper
pip install -r requirements.txt
cp .env.example .env
```
Заполни в `.env`:
```dotenv
BOT_TOKEN=токен
ADMIN_IDS=твой_id
MARKETS=dmarket,steam          # или mock — для теста без денег
DMARKET_PUBLIC_KEY=...
DMARKET_SECRET_KEY=...
# защита (значения по умолчанию, потом меняются в боте):
AUTOBUY=0
DRY_RUN=1
MAX_ITEM_PRICE=50
DAILY_LIMIT=100
MIN_BALANCE=0
```

### 4. Запуск
```bash
python run.py
```
Открой бота → `/start`.

## 🕹 Как пользоваться

1. **➕ Добавить правило** — мастер спросит: название (напр. `Case Hardened`),
   макс. цену, диапазон float (напр. `0`–`0.07` = Factory New), редкие seed
   (бот подскажет известные), наклейки, мин. скидку, и покупать ли автоматически.
2. Бот начинает слать **🔔 алерты** по совпадениям.
3. Кнопки-переключатели в меню:
   - **🔎 Монитор** — вкл/выкл слежение.
   - **🛒 Автобай** — глобально разрешить авто-покупку.
   - **🧪 Dry-run** — пока ВКЛ, реальные деньги не тратятся (только «купил бы»).
4. **💰 Баланс** и **🧾 Покупки** — контроль.

### ✅ Безопасный порядок запуска
1. `MARKETS=mock` → проверь, что алерты и «покупки» приходят (без денег).
2. Переключись на `dmarket,steam`, оставь **Dry-run ВКЛ**, добавь правила —
   смотри, что бот ловит правильные лоты.
3. Убедился — выключи Dry-run и включи Автобай. Начни с **низких лимитов**
   (`MAX_ITEM_PRICE`, `DAILY_LIMIT`) и небольшого баланса.

## 🖥 Запуск 24/7 (VPS)

Так же, как обычный Python-бот: `venv` + `systemd`. Кратко:
```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
# .env заполнен
```
systemd-юнит (`/etc/systemd/system/cs2-sniper.service`):
```ini
[Unit]
Description=CS2 Skin Sniper
After=network-online.target
Wants=network-online.target
[Service]
WorkingDirectory=/opt/sniper
ExecStart=/opt/sniper/.venv/bin/python run.py
Restart=always
RestartSec=5
[Install]
WantedBy=multi-user.target
```
```bash
sudo systemctl daemon-reload && sudo systemctl enable --now cs2-sniper
journalctl -u cs2-sniper -f
```
Подойдёт VPS 1–2 vCPU / 1–2 ГБ RAM / 20 ГБ SSD (мониторинг — лёгкий).

## ⚠️ Важно и честно

- Авто-покупка на DMarket идёт через официальный API и тратит **твои** деньги.
  Эндпоинт покупки/подписи DMarket сверяй с актуальной докой (docs.dmarket.com);
  бот по умолчанию в Dry-run, чтобы ты сначала всё проверил.
- **Steam Market** — только алерты. Автоматизировать покупки на Steam нельзя без
  нарушения правил Steam и риска бана.
- Снайпинг — конкурентная штука: такие же боты крутят и другие, площадки могут
  ограничивать частоту запросов (`POLL_INTERVAL`). Это не «печатная машинка», а
  инструмент, повышающий шансы.

Тесты: `pip install pytest pytest-asyncio && pytest`.
