# Демо Tythan Code — сценарий на 5 минут

Эта папка — заранее подготовленный «сломанный» мини-проект, на котором все
функции Tythan Code показывают себя максимально эффектно. Файлы содержат
**намеренные** уязвимости и баги — не используйте их в продакшене.

## Подготовка (1 раз, до встречи)

1. Установите расширение: `code --install-extension tythan-code-*.vsix`
2. Откройте **эту папку** (`demo/`) как отдельное окно VS Code: `code demo/`
3. Задайте ключ: `Ctrl+Shift+P` → **Tythan Code: Set API Key for Provider**
4. Проверьте, что чат отвечает (иконка Tythan Code в левой панели).

## Сценарий демо

### 1. Security-аудит — «вау» за 20 секунд (наша главная фишка)

`Ctrl+Shift+P` → **Tythan Code: Run Security Audit**

Скажите: *«Ни один запрос никуда не ушёл — сканер работает офлайн. SQL-инъекция,
захардкоженный пароль, выключенный TLS, debug-режим — всё найдено до коммита,
а не после взлома»*. Cursor такого из коробки не делает.

### 2. Чат-агент чинит найденное

В чате: `Исправь все CRITICAL и HIGH проблемы в @app.py`

Покажите: агент читает файл, на **каждую** правку открывается настоящий дифф
VS Code, вы жмёте Apply. В конце: **Tythan Code: Undo Last Agent Change** —
и всё откатилось одним действием. Снова Redo? Просто повторите запрос.

### 3. Composer — мультифайловая задача (Ctrl+Alt+I)

Задача для Composer:
`Добавь в app.py эндпоинт /health, вынеси конфигурацию в config.py и обнови README`

Покажите: агент правит несколько файлов, но **диск не тронут** — в конце
список файлов с чекбоксами, подсветка показывает дифф каждого. Снимите
галочку с README — применится только остальное. Один Undo откатывает всё.

### 4. Быстрые инструменты (по 15 секунд каждый)

- **Ctrl+Alt+K** на выделенной функции в `utils.js`: «перепиши без багов и
  добавь JSDoc» → дифф → Apply.
- Наведите на подчёркнутую ошибку → 💡 → **Fix with Tythan Code**.
- `@codebase где считается цена со скидкой?` в чате — сам найдёт нужный код.
- Внесите правку, откройте Source Control → кнопка ✨ — готовый commit message.

### 5. Финальный аргумент

Откройте `.tythanrules`: *«Правила проекта — агент им следует в каждом ответе.
Ваши ключи хранятся в системном хранилище VS Code, код на индексацию никуда
не отправляется, каждая правка — через подтверждаемый дифф с полным undo.
Это Cursor-workflow, но security-first и с любой моделью — включая локальную,
когда код нельзя выносить за периметр»*.
